# Venue Rental – Frontend

Venue Rental is a web application that allows users to browse and book event venues through a clean and intuitive interface. The frontend was designed with a strong focus on usability, visual hierarchy, and responsive design.

This repository contains the **frontend** of the application, built to deliver a polished user experience across desktop and mobile devices.

## Features
- Responsive homepage showcasing featured venues
- Detailed venue pages with images, pricing, and descriptions
- Booking flow with user dashboard
- Modern, minimalist UI design

## Tech Stack
- **React** – Component-based UI development
- **Tailwind CSS** – Utility-first responsive styling
- **JavaScript (ES6+)**

## Design Process
- Wireframing and prototyping using **Figma**
- Focus on layout consistency, spacing, and accessibility
- Iterative design improvements based on usability principles
